# Cinelog
An app to store your movie watchlist and reviews
## Set up

### Backend
+ Ensure you have your database running in backgroud
+ Copy the ./backend/.env.example to create ./backend/.env file 
+ Install deps using 
```
npm i
```
+ Start backend by executing
```
npm run dev
```

### Frontend
+ Ensure the backend is running
+ Edit the URL in ./frontend/.env.local to point to your local backend
+ Fire up frontend using
```
npm run dev
```
